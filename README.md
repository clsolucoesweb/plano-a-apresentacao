# Plano A | Apresentação Institucional

Apresentação web estática da Plano A, administradora de planos de saúde, feita para publicar diretamente no GitHub Pages.

## Como visualizar localmente

Abra `index.html` no navegador ou rode um servidor local:

```bash
python3 -m http.server 4173
```

Depois acesse `http://localhost:4173`.

## Navegação

- Setas esquerda/direita
- Page Up/Page Down
- Home/End
- Botões laterais
- Miniaturas na barra inferior

## Publicar no GitHub Pages

1. Envie estes arquivos para um repositório no GitHub.
2. Entre em `Settings > Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Selecione a branch `main` e a pasta `/root`.
5. Salve e aguarde a URL pública do GitHub Pages.

Como é um site estático, não precisa de build.
